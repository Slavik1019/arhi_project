Product (Товар)

Описание структуры ER диаграммы для приложения по формированию отчетов на основе разных данных деятельности компании

Это сущность, представляющая информацию о товарах.
У товара есть следующие атрибуты: ProductID (тип данных: целое число), Name (тип данных: строка), Category (тип данных: строка), и Quantity (тип данных: целое число).
Товары связаны с элементами продажи (SaleItem) через атрибут ProductID.
Также товары могут храниться на складе (Stock), связанным через атрибут ProductID.

Counterparty (Контрагент)

Сущность, представляющая информацию о контрагентах (поставщиках или клиентах).
Атрибуты: CounterpartyID (целое число), Name (строка), Address (строка), и Phone (строка).
Контрагенты связаны с продажами через SaleCounterparty.

Sale (Продажа)

Это сущность, представляющая информацию о продажах.
Атрибуты: SaleID (целое число), SaleDate (дата), и Amount (вещественное число).
Продажи связаны с товарами через элементы продажи (SaleItem) и с контрагентами через контрагентов продажи (SaleCounterparty).
Продажи связаны с финансовыми показателями через финансовые элементы продажи (SaleFinancialItem).

Stock (Склад)

Сущность, представляющая информацию об остатках товаров на складе.
Атрибуты: StockID (целое число), ProductID (целое число), и Quantity (целое число).
Склад связан с товарами через атрибут ProductID.

Financial Indicator (Финансовый показатель)

Сущность, представляющая информацию о финансовых показателях.
Атрибуты: IndicatorID (целое число), Name (строка), и Value (вещественное число).
Финансовые показатели связаны с продажами через SaleFinancialItem.

User (Пользователь)

Это сущность, представляющая информацию о пользователях системы.
Атрибуты: UserID (целое число), Login (строка), и Password (строка).
Пользователи связаны с подключениями к базе данных (Database Connection).
Database Connection (Подключение к базе данных)

Сущность, представляющая информацию о подключениях к базе данных.
Атрибуты: ConnectionID (целое число), DatabaseName (строка), и IsConnected (логический тип).
Подключения к базе данных связаны с пользователями (User).

Report (Отчет)

Сущность, представляющая информацию о созданных отчетах.
Атрибуты: ReportID (целое число), Name (строка), ReportType (строка), и Content (строка).
Отчеты связаны с генераторами отчетов (Report Generator).

Report Generator (Генератор отчетов)

Сущность, представляющая информацию о генераторах отчетов.
Атрибуты: GeneratorID (целое число), Name (строка), и Description (строка).
Генераторы отчетов связаны с отчетами (Report).

SaleItem (Элемент продажи)

Сущность, представляющая информацию о конкретных товарах внутри каждой продажи.
Атрибуты: SaleItemID (целое число), SaleID (целое число), ProductID (целое число), и Quantity (целое число).
Элементы продажи связаны с продажами (Sale) и товарами (Product).

SaleCounterparty (Контрагенты продажи)

Сущность, связывающая продажи (Sale) с контрагентами (Counterparty).
Атрибуты: SaleCounterpartyID (целое число), SaleID (целое число), и CounterpartyID (целое число).
Позволяет отслеживать, с какими контрагентами связаны продажи.

SaleFinancialItem (Финансовые элементы продажи)

Сущность, связывающая продажи (Sale) с финансовыми показателями (Financial Indicator).
Атрибуты: SaleFinancialItemID (целое число), SaleID (целое число), и FinancialIndicatorID (целое число).
Позволяет отслеживать финансовую информацию, связанную с продажами.
